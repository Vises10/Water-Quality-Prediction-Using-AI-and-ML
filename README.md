# Water-Quality-Prediction-Using-AI-and-ML

Water Quality Prediction System
An AI/ML-based project that predicts whether water is safe for drinking (potable) or not, based on various chemical properties.

Features->

Predict water potability (Safe / Unsafe)
Machine Learning based prediction
Data analysis and preprocessing
Model comparison using graphs
Simple and easy to understand implementation

How It Works->

Collects water quality dataset
Cleans and preprocesses the data
Trains multiple machine learning models
Evaluates their performance
Compares results using visualization
Predicts whether water is potable or not
